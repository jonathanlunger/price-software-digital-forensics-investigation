# Price Software Digital Forensics Investigation

## Overview

This repository contains a digital forensic case study involving suspected insider activity, unauthorized access, and possible exposure of proprietary company information.

The investigation examined forensic images from employee workstations, a company server, and a reconstructed RAID 5 array. Findings were correlated across email artifacts, browser activity, financial documents, deleted files, network-share shortcuts, and removable-media artifacts.

## Investigation Objectives

The examination focused on determining whether:

- Proprietary company information was accessed or exposed
- Employee activity could be attributed to specific user accounts
- Personal webmail was used for outside communication
- Removable media or administrative shares may have supported data movement
- Deleted artifacts or secure-deletion tools indicated possible concealment

## Tools Used

- AccessData Forensic Toolkit 8.2
- X-Ways Forensics
- Avast Antivirus
- MD5 hash verification

## Key Skills Demonstrated

- RAID 5 reconstruction
- Disk image examination
- Windows registry analysis
- Email and webmail artifact analysis
- Browser history analysis
- Deleted file and Recycle Bin analysis
- User attribution
- Timeline reconstruction
- Evidence correlation
- Formal forensic reporting
